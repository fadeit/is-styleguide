Colors

The colors in this list are the only colors to be used.

Conventions

  • Color variables are prefixed with col.
  • Darker, contrasting versions of a color are sufixed with _c: @w (white) and @w_c (white contrast)

Color list

Colors are defined in variables.less. A full list can also be found there.

Typography

Use heading 1 (h1) just once per page.
The measure on the left represents font size in pixels, the second line height. (30/40 pixels = 30px font size, 40px font size). Keep in mind that heading 4 and 5 are in bold.
The background in the example is not in the styleguide, the headings have no background by default.

LESS variables are defined as @fs_[heading-number] for font-size (fs) and @lh_[heading-number] for line-heights (lh). Examples: @fs_h1, @lh_h1.

Scaffolding: Layout, Sections & Grid system

The grid system is made out of 24 columns and works exactly as the Bootstrap Grid, times 2. The grid double-up is chosen for flexiblity. In markup, it just means doubling up all the CSS column sizes.

Normal page

Pages are made out of multiple rows, each row containing 24 columns. For mobile devices (col-xs), the column will take 100% of the width (all 24 columns).

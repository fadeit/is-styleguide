Colors

The colors in this list are the only colors to be used.

Conventions

  • Color variables are prefixed with col.
  • Darker, contrasting versions of a color are sufixed with _c: @w (white) and @w_c (white contrast)

Color list

Heads up! Colors are defined in variables.less. The same list can be seen there.
Using Sublime Text? Get the Color Highlighter package to see the colors in your favorite text editor.
